# Oracle 11g 分页规则

---

## ORACLE-11G-PAGINATION-001：必须用 ROWNUM，禁用 OFFSET FETCH

**严重性**：🔴 严重

Oracle 11g **不支持** `OFFSET ... FETCH` 语法（12c 引入）。必须用 ROWNUM 子查询。

```sql
-- ❌ Oracle 11g 报错
SELECT * FROM orders ORDER BY id OFFSET 20 ROWS FETCH NEXT 10 ROWS ONLY;

-- ✅ Oracle 11g 正确写法
SELECT * FROM (
    SELECT t.*, ROWNUM rn
    FROM (
        SELECT * FROM orders ORDER BY id
    ) t
    WHERE ROWNUM <= 30
) WHERE rn > 20;
```

**分页模板**（pageNum 从 1 开始，pageSize 每页条数）：
```sql
SELECT * FROM (
    SELECT t.*, ROWNUM rn
    FROM (SELECT * FROM table_name ORDER BY sort_col) t
    WHERE ROWNUM <= #{pageNum} * #{pageSize}
) WHERE rn > (#{pageNum} - 1) * #{pageSize}
```

**证据**：Oracle 11g SQL Reference §12.29（OFFSET FETCH 在 12c 引入）
