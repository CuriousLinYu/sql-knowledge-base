# Oracle 19c 分页规则

Oracle 19c 同时支持 ROWNUM 子查询和标准 `OFFSET FETCH` 语法。

---

## 推荐：OFFSET FETCH（12c+）

```sql
-- 简洁标准写法
SELECT * FROM orders ORDER BY id
OFFSET 20 ROWS FETCH NEXT 10 ROWS ONLY;
```

## 兼容：ROWNUM 子查询（所有版本通用）

```sql
-- 兼容 11g 及以下
SELECT * FROM (
    SELECT t.*, ROWNUM rn
    FROM (SELECT * FROM orders ORDER BY id) t
    WHERE ROWNUM <= 30
) WHERE rn > 20;
```
